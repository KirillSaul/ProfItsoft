## Run project:
In navbar choose "Run"->Edit configurations->"+"->Tomcat Server->local:
- Page "Server":
    - Application server - set path to your tomcat
    - URL: http://localhost:8080/ (or any other port)
- Page "Deployment":
    - Create your .war file
    - Make empty "Application context"
